# APB-Based-SPI-Master-Interface
Designed an APB SPI Master Core which includes the following modules: APB Slave Interface, Baud Rate Generator, Slave Select Controller, Shift Register and Top Order Level. This project was performed in ModelSim and Quartus Prime. In ModelSim, the testbenches were verified with different modes of SPI operation and checked their functionality. In Quartus Prime, synthesis analysis of RTL Design was performed of all the individual blocks as well as the top order block.
